1.Потребуется среда VS или Android Studia с расширениями: Dart/Flutter
2.Необходимо выгрузить проект на компьютер
два пути:
------Консоль
создайте директорию на рабочем столе(так удобно)
в gitBush(эта консоль у нас) введите команду cd desktop
если через обычную консоль - дойдите до этой папки через последовательность команд cd

затем пропишите команду git clone https://github.com/SkibaSAY/WAY.git
далее откройте полуенный каталог через редактор кода
зайдите в main и запустите программу(по стандарту откроется в гугл)
-----Просто скопируйте файлы из репозитории на компьютер и откройте их в редакторе
3.Запустите проект, дождитесь завершения запуска эмулятора или сервисов Google
